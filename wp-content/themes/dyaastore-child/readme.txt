== Fitur Custom ==

1. Custom field "Username Roblox" pada halaman checkout WooCommerce
2. Validasi format username Roblox (3-20 karakter, alphanumeric + underscore)
3. Tampilan username Roblox di:
   - Halaman detail order (admin)
   - Email konfirmasi (customer & admin)
   - Halaman "Thank You" setelah checkout
4. Auto-set produk baru sebagai Virtual (untuk produk digital)
5. Branding admin (judul tab + footer)

== Instalasi ==

1. Pastikan plugin WooCommerce sudah aktif
2. Pastikan tema parent "Hello Elementor" sudah terinstall
3. Aktifkan tema ini via Appearance > Themes
