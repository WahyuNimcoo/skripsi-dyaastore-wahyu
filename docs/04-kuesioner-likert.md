## Cara Pengolahan Data (untuk BAB IV Skripsi)

### Rumus Persentase
```
Persentase = (Total Skor / Skor Maksimum) × 100%
```

Dimana:
- Skor maksimum = `Jumlah Responden × Jumlah Pertanyaan × 5`
- Total skor = `Σ (skor jawaban responden)`

### Interpretasi Hasil

| Persentase | Kategori |
|------------|----------|
| 0% – 20% | Sangat Tidak Baik |
| 21% – 40% | Tidak Baik |
| 41% – 60% | Cukup |
| 61% – 80% | Baik |
| 81% – 100% | Sangat Baik |

### Contoh Perhitungan
```
10 responden × 16 pertanyaan = 160 jawaban
Skor maksimum = 160 × 5 = 800
Total skor = 680
Persentase = (680 / 800) × 100% = 85% → "Sangat Baik"
```
